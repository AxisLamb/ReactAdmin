# ReactAdmin
最新版本的经典的RBAC后台管理系统，后端使用JDK21 + SpringBoot3.2.0；前端使用NodeJS v22.17.0 + Ant Design React 5.x。The latest version of the classic RBAC (Role-Based Access Control) backend management system, with the backend using JDK 21 + Spring Boot 3.2.0, and the frontend using Node.js v22.17.0 + Ant Design React 5.x.
